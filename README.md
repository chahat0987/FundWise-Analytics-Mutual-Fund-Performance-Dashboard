#  Mutual Fund Analysis & Dashboard

A complete data analytics project focused on identifying high-performing mutual funds using Python, Excel, and Power BI.

##  Project Overview

This project analyzes mutual fund schemes to identify funds with strong returns, lower risk, and better investment potential. The analysis combines data cleaning, scoring, ranking, and visualization techniques to generate actionable investment insights.

###  Objective

* Analyze mutual fund performance data
* Identify top-performing schemes based on returns and risk
* Build an interactive dashboard for investment insights
* Demonstrate end-to-end data analytics workflow
